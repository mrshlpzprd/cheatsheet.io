HTML Elements Reference

Main root

Tag Description
<html> Represents the root (top-level-element) of an HTML document

Document metadata

Tag Description
<head> Contains metadata about the document, such as title, scripts and style sheets
<link> Defines the relationship between a document and an external resource (most used to link to style sheets)
<style> Defines style information for a document
<title> Defines a title for a document

Sectioning root

Tag Description
<body> Defines the document's body. There can be only one <body> element in a document

Content sectioning

Tag Description
<address> Defines contact information for the author/owner of a document
<article> Represents a self-contained composition which is intended to be independently distributable or reusable. Examples: a forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted content, an interactive widget or gadget, or any other independent item of content
<footer> Defines a footer for a document or section. It typically contains information about the author of the section, copyright data or links to related documents
<header> Represents introductory content, typically a group of introductory or navigational aids
<h1> to <h6> Represents six levels of section headings
<nav> Defines navigation links
<section> Represents a generic standalone section of a document

Text content

Tag Description
<div> Is the generic container for flow content
<figcaption> represents a caption or legend describing the rest of the contents of its parent figure element
<li> Used to represent an item in a list. It must be contained in an ordered list (ol) or an unordered list (ul)
<p> Represents a paragraph

Inline text semantics

Tag Description
<a> Defines a hyperlink
<code> Defines a piece of computer code
<span> Is a generic inline container for phrasing content. It's an inline element. It should be used only when no other semantic element is appropriate

Image and multimedia

Tag Description
<audio> Defines embedded sound content
<img> The Defines an image
<video> The Defines embedded video content
